# Swiss Skull

Настольное приложение на Python для обработки медицинских изображений, специально разработанное для сегментации и анализа МРТ-снимков. Приложение имеет графический интерфейс, построенный на PySide6 для интерактивной визуализации и обработки изображений.

## Требования

- Python 3.11+
- Пакетный менеджер [uv](https://github.com/astral-sh/uv)

Этот проект зависит от нескольких научных и визуализационных библиотек, включая:

- PySide6 (фреймворк GUI)
- PyVista (3D визуализация)
- NumPy, SciPy (численные вычисления)
- Matplotlib (построение графиков)
- VTK (Visualization Toolkit)
- scikit-image (обработка изображений)
- ImageIO (ввод-вывод изображений)

Полный список зависимостей см. в `pyproject.toml`.

## Установка

### 1. Установите uv (если еще не установлен)

MacOS/Linux:

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

Windows:

```bash
powershell -c "irm https://astral.sh/uv/install.ps1 | iex"
```

### 2. Синхронизируйте зависимости

```bash
uv sync
```

Это автоматически создаст виртуальное окружение и установит все зависимости из `pyproject.toml`.

## Запуск приложения

```bash
uv run main.py
```

Откроется окно приложения с графическим интерфейсом для работы с инструментами сегментации и анализа.

## Структура проекта
  - `main.py` - точка входа, запускающая GUI
  - `segmentation/` - Основные алгоритмы сегментации и компоненты интерфейса (включая `MainWindow`)
  - `convertation/` - Утилиты для преобразования данных
- **`pyproject.toml`** - Метаданные проекта и спецификация зависимостей Python
